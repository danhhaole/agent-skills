## Story structures

These are scaffolds, not cages (Aristotle's first principle). Pick one that fits
the story's shape; don't force a story onto a model.

### Three-act structure
- **Act 1 — Setup (~25%):** ordinary world, protagonist's want and flaw, then the
  **inciting incident** that disrupts it. Ends on a choice that locks them in.
- **Act 2 — Confrontation (~50%):** escalating obstacles; a **midpoint** that
  raises stakes or flips understanding; the **all-is-lost** low point at ~75%.
- **Act 3 — Resolution (~25%):** the climax that tests the lesson, then the new
  equilibrium. The ending answers the dramatic question Act 1 posed.

### Save the Cat (15 beats)
Opening Image → Theme Stated → Set-Up → Catalyst → Debate → Break into Two →
B Story → Fun and Games → Midpoint → Bad Guys Close In → All Is Lost →
Dark Night of the Soul → Break into Three → Finale → Final Image.
Useful for tight commercial pacing; the percentages above map onto it.

### Hero's Journey (condensed)
Ordinary World → Call to Adventure → Refusal → Meeting the Mentor → Crossing the
Threshold → Tests, Allies, Enemies → Approach → Ordeal → Reward → The Road Back →
Resurrection → Return with the Elixir. Strong for myth, fantasy, coming-of-age.

### Kishōtenketsu (4 acts, conflict-optional)
Ki (intro) → Shō (development) → Ten (twist/unexpected turn) → Ketsu
(reconciliation). Useful for literary, slice-of-life, or stories driven by
revelation rather than conflict.

### Scene-level: value change
Every scene should turn a charged value (safe→threatened, hope→despair,
love→betrayal). If the value is the same at the end as the start, the scene is
probably inert — Aristotle's `scene audit` cuts or fixes it.

## Genre conventions

Each genre is a *promise* readers came for — honour or subvert it deliberately,
never break it by accident (Borges' rule). The full per-genre breakdown — promise,
a `genre check` checklist, and common pitfalls for fantasy, SF, mystery, thriller,
romance, horror, literary, YA, and historical — lives in **`references/genres.md`**.
Read that file when setting expectations at Phase 0 or running a genre QA pass.

## Elicitation moves

When an author wants to develop a section further, offer ~4 moves as a numbered
list, apply the chosen one from the current member's perspective, then re-offer
until they pick "proceed." The goal is to draw the *author's* ideas out, not to
overwrite them.

**Choose moves that fit what's on the table** — don't paste the same four every
time. Read the content type and offer the set that will actually push *this*
section. The grouped menus below are starting points; mix in a move from another
group when it's clearly the most useful one. Always include "proceed" as the last
option.

**Structure / outline** (Aristotle)
- **Raise the stakes** — what makes this matter more, or cost more?
- **Causality check** — is this beat *because of* the last, or just *and then*?
- **Twist it** — invert an assumption; what if the opposite were true?
- **Comp it** — how do comparable published books handle this beat?

**Character** (Fyodor)
- **Go deeper** — push on the wound, the lie, the contradiction beneath the want.
- **Want vs. need** — do the external goal and internal truth pull against each other?
- **Bolder alternative** — a riskier, less expected version of this person.
- **Pressure-test** — drop them in a hard choice; does the psychology hold?

**World** (Tolkien)
- **Cost & limit** — what does this rule/magic/tech *cost*? Where does it stop?
- **Consequence** — follow the premise out: what does this world imply, socially?
- **Sensory specificity** — one concrete, telling detail over a general description.
- **Consistency check** — does this contradict anything already in the bible?

**Scene / prose** (Scheherazade, Max)
- **Go deeper** — sharpen motivation, stakes, or sensory detail.
- **Value turn** — what charged value changes from the scene's start to its end?
- **Enter late / leave early** — can this open later and cut off sooner?
- **Show vs. tell** — dramatize the named emotion through action/sensation.

**Dialogue** (Oscar)
- **Subtext** — make them say less than they mean; what's under the line?
- **Distinct voice** — could we tell who's speaking with the tags removed?
- **Double duty** — can this exchange advance plot *and* reveal character at once?
- **Compress** — cut the throat-clearing; start the exchange mid-tension.

**Cross-cutting** (any member, when relevant)
- **Critique & refine** — name the weakest part and propose a fix.
- **Red-team** — argue why a reader won't buy this; then address it.
- **Genre lens** — how does this read to a fan of the genre? (Borges)
- **Reader reaction** — how does this *feel* to read, beat by beat? (Virginia)
- **Proceed** — looks good, continue to the next section.
