# Character Profile — `characters/<name>.md`

Phase 2, led by Fyodor. One file per major character. The psychological core
(section 2) matters most — a vivid wound/lie/want/need does more work than pages
of biography. Offer elicitation after the psychology and arc sections especially.

## Sections

### 1. Basics
Name and nicknames · age · role in the story · first impression · a one-line
physical detail that *means* something (not a passport description).

### 2. Psychological core
The engine of the character:
- **Wound / ghost** — the past hurt that still shapes them
- **Lie they believe** — the false idea about themselves or the world the wound
  taught them
- **Want** — the external goal they chase (drives plot)
- **Need** — the internal truth they must learn (drives arc); it usually
  *contradicts* the want
- **Fear** — what they'll do almost anything to avoid
- **Defense mechanisms** — how the wound shows up in behaviour

### 3. Backstory (only what feeds the story)
Formative events, key relationships, secrets held. Keep it lean — backstory
informs behaviour; it is not a biography to be recited.

### 4. Voice
Vocabulary and rhythm · what they avoid talking about · how they argue · humour ·
two or three sample lines that only this character would say. (Oscar uses this in
the dialogue pass.)

### 5. Relationships
Bonds, debts, rivalries, and power dynamics with other characters.

### 6. Arc
Starting state → inciting impact → resistance → turning point → dark moment →
breakthrough → end state. This must align with the arc sketched in `outline.md`.

### 7. Telling details
A habit, a contradiction, a prized possession — the specifics that make them feel
like a person, not a function.
